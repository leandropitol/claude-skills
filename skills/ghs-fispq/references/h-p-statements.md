## Declarações de Perigo (H-statements)

Estrutura dos códigos:
- Letra **H** + número de 3 dígitos
- **H2xx** = perigos físicos
- **H3xx** = perigos à saúde
- **H4xx** = perigos ambientais

## Frases H mais comuns na indústria química (referência rápida)

Para produtos **inflamáveis** (terebintina, solventes):
H224 (Pf<23°C, PE≤35°C) → H225 (Pf<23°C) → H226 (23≤Pf≤60°C) → H227 (60<Pf≤93°C)
+ H304 (aspiração) se viscosidade cinemática ≤20,5 mm²/s a 40°C

Para **colofônia/breu** (sensibilizantes):
H317 (sensibilizante cutâneo Cat.1) + avaliar H412/H413 (perigo aquático crônico)

Para **produtos corrosivos** (ácidos/bases):
H290 (metais) + H314 (pele/olhos Cat.1) ou H315+H318/H319 (Cat.2+Cat.1/2)

Para **tóxicos por inalação**:
H330 (Cat.1/2) → H331 (Cat.3) → H332 (Cat.4) + avaliar H335/H336 (STOT Cat.3)
