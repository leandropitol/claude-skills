# Checklist de Auditoria — FISPQ / Rótulo GHS

## Como usar este checklist

Para cada não conformidade encontrada, classificar como:
- **NC Crítica** (❌): impede o uso do documento; risco imediato
- **NC Maior** (⚠️): compromete conformidade normativa; corriger antes da próxima distribuição
- **Observação** (ℹ️): melhoria de qualidade ou boas práticas

## Modelo de relatório de auditoria

```
RELATÓRIO DE AUDITORIA GHS/FISPQ
Produto: [nome]
Data: [data]
Auditor: [nome/referência]
Versão auditada: [versão da FISPQ]

SUMÁRIO EXECUTIVO:
[ ] ✅ Conforme — documento apto para uso
[ ] ⚠️ Com ressalvas — corriger antes da próxima distribuição
[ ] ❌ Não conforme — suspender uso; emitir nova versão

NÃO CONFORMIDADES CRÍTICAS (❌):
1. [NC + seção + descrição + ação requerida]

NÃO CONFORMIDADES MAIORES (⚠️):
1. [NC + seção + descrição + prazo sugerido]

OBSERVAÇÕES / OPORTUNIDADES DE MELHORIA (ℹ️):
1. [observação + seção + sugestão]

RECOMENDAÇÕES PRIORITÁRIAS:
1. [ação + responsável + prazo]
```
