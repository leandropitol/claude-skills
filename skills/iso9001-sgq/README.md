# iso9001-sgq

**Skill:** Sistema Decisório Multi-Agente para ISO 9001:2015  
**Domínio:** Gestão da Qualidade / SGQ  
**Empresa:** RJ Comércio e Extração de Resinas Ltda. (Resinas Jardim)

---

## Descrição

Sistema multi-agente especializado em ISO 9001:2015, projetado para suportar a implementação e manutenção do Sistema de Gestão da Qualidade (SGQ) da Resinas Jardim. Orquestra 8 agentes especializados, cada um responsável por um domínio específico da norma.

## Agentes Incluídos

| Agente | Domínio |
|--------|---------|
| Analista SGQ | Coordenação geral, não conformidades, CAPA |
| Representante da Direção | Política, objetivos, análise crítica |
| Planos de Ação | 5-Porquês, Ishikawa, 8D, FMEA |
| Documentos e Registros | Controle de documentos, lista mestra |
| Planejamento Estratégico | SWOT, partes interessadas, contexto |
| Gestão de Riscos | Matriz de risco, plano de mitigação |
| Auditor Interno | Checklist, relatório de auditoria |
| Metrológica | Calibração, confirmação metrológica |

## Gatilhos

Acione esta skill ao mencionar: ISO 9001, SGQ, não conformidade, NC, ação corretiva, RAC, RNC, CAPA, 8D, causa raiz, 5-Porquês, Ishikawa, FMEA, auditoria interna, checklist de auditoria, relatório de auditoria, controle de documentos, lista mestra, gestão de riscos, matriz de risco, calibração, confirmação metrológica, política da qualidade, objetivos da qualidade, análise crítica pela direção, PDCA, melhoria contínua, produto não conforme, indicadores de qualidade, KPI de qualidade.
